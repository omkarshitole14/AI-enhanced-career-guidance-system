# AI Enhanced Career Guidance System

## Overview

AI Enhanced Career Guidance System is a Machine Learning based web application that predicts suitable career paths using:

- Education Level
- Specialization
- Skills
- Certifications
- CGPA
- Resume PDF

The system also performs:

- Resume Parsing
- Skill Gap Analysis
- Match Score Calculation
- Career Roadmap Generation
